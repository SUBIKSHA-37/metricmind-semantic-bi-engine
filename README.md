# MetricMind – Agentic Semantic BI Engine

## Overview

MetricMind is an enterprise-grade conversational Business Intelligence platform that combines:

- Semantic Layer
- Agentic AI
- Data Warehouse
- Interactive Dashboard

Instead of allowing an LLM to generate raw SQL, MetricMind queries governed business metrics through a semantic layer, ensuring trustworthy and consistent analytics.

## Tech Stack

- Snowflake
- dbt
- Cube.dev
- LangChain
- Llama 3
- Next.js
- Tremor
- ECharts
